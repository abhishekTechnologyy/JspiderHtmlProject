Marquee tags

  • This tag is used to create scrolling contents on the web page
  • Attributes of MArquee tags are :
    Beahaviour
    • It defines the type of scroll of the elements
    • scroll , alternate and slide
    • Scroll is default
    Direction
    • It specifies the direction in which the elements are scrolling
    • Right,Left,Up and Down
    • Left is Default
    Scroll amount
    • It controls the speed of scroll
    • The speed increase if the value of scroll amount increase
    Height and Width
    • Used to adjust the height and width of marquee
