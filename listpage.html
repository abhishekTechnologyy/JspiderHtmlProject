Lists

Lists are used to display similar set of items on a webpage.
Types of List:

  1. Unordered List (ul)
  2. Ordered List (ol)
  3. Description List (dl)

UL

  1. It is used to display the list items, where there is no priority for the order.
  2. By default all the list items will be displayed with the help of bullets (disc)
  3. The style can be changed using "type" attribute.

    Types are :

    1. disc (default)
    2. square
    3. circle

OL

  1. It displays the list items where the priority is given for the order
  2. ordered list will be display all the elements with the help of numbers, which can be changed using the "type attributes.

    Types are :

    1. Numbers (default)
    2. A
    3. a
    4. I
    5. i

Start Attribute

  1. It specifies the start index of the ordered lists
  2. It only takes nu,bers as input.


DL

  1. HTML uses < dl> tag to describe a list of items.
  2. < dt> tags are used to display the term and the definition for the terma are writtenusing < dd> tag.
