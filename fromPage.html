Forms

HTML uses "forms" tag to collect the user input that a user should enter.


  1. < label> tag :

    It is used to display the type of input that a user should enter.

  2. < input> tag

    It is used to collect different types of user input.


    Attributes for input tag

    Type :
    1. It specifies the type of value from a user.
    2. text , email , password , date , number , tel , checkbox , radio , file
    Name :
    This attribute specifies where the input values will be stored in the backend
    Required :
    "req" it is used to make input field mandatory.
    Place Holder:
    It displays a text within the input field
    min and max :
    It is used along with input type "number" that specifies the minimum and the maximum value that a user should enter
    minlength and axlength :
    It is used to set a limit for minimum and maximum no. of characters that a user should enter.
    Text area :
    This is used to enter larger input. The size of the text area can be controlled using rows and columns values
    Select tag :
    It is used to create a drop down for multiple options.
